# Comparison-of-two-scenes

Test task: compare two point cloud scenes with Potree.js.

## Commands

- `npm run check` - validate project integrity and required dependencies.
- `npm run start` - start the local server at `http://localhost:3000`.

## Note

The repository does not include point cloud datasets yet, so the start page initializes a Potree viewer and shows a message that data still needs to be added separately.
